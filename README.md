## 📝 Project Overview
Developed in **2023**, ini adalah proyek fondasi saya dalam pengembangan aplikasi *mobile*. Fokus utamanya adalah mendigitalisasi katalog fisik suku cadang sepeda motor menjadi sistem digital *real-time* yang portabel. 

Aplikasi ini dirancang khusus untuk memfasilitasi tim lapangan dan staf gudang dalam mengakses data produk, verifikasi detail item, serta manajemen input inventaris secara instan.

---

## 🛠️ Technical Stack
*   **Framework:** Flutter
*   **Language:** Dart
*   **Core Focus:** Scalable UI/UX Architecture, State Management, and Navigation Workflows.

---

## 🚀 Key Features

*   **📦 Warehouse Catalog (Gudang):** Tampilan berbasis *grid* untuk menampilkan suku cadang, gambar, kode barang, dan harga.
*   **🔍 Item Detail Insights:** Modul interaktif (*modals*) untuk data spesifik seperti Kode Barang, Satuan, dan Persamaan.
*   **📥 Inventory Input:** Formulir pendaftaran item baru ke dalam *registry* digital secara efisien.
*   **🗺️ Multi-Module Navigation:** Menu *drawer* terstruktur untuk navigasi antar modul yang mulus.
